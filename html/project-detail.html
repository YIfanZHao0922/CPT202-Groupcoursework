Available

Supervisor: Dr. John Smith | Department of Computer Science

Project Overview

This project aims to develop a full-stack web application for managing student project topic selection, application workflow, and supervision tracking. The system will support three distinct user roles: administrator, supervisor, and student, each with tailored access permissions and functionalities.

Project Objectives

  • To design and implement a user-friendly interface for project topic browsing and selection
  • To develop a secure role-based access control system for different user types
  • To create an automated workflow for project application submission, review, and decision-making
  • To implement a database system for efficient project and user data management

Required Skills

HTML5 CSS3 JavaScript SQL Web Development Database Design

Project Scope & Deliverables

  • Complete system architecture design and documentation
  • Fully functional front-end and back-end system
  • Comprehensive testing report and user manual
  • Final project demonstration and presentation
